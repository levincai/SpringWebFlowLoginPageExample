
This application is a basic spring webflow application thats shows the following concepts:

1) How to integrate spring mvc with spring webflow.
2) How you can validate user input through validation logic in your model object
3) How you can validate user input in a method in action class. 
4) How to call another flow as a subflow
5) How to pass a value to subflow
6) How you can test flow executions.


Used Spring 3 MVC and Spring Webflow 2.3 for this sample app.
